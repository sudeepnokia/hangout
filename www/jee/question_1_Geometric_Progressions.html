Build Your Skills: Geometric Progressions

This guide will build your understanding from the ground up, so you can solve a real JEE Main problem on your own.

Phase 1: What is a Sequence?

In math, a sequence is just a list of numbers in a specific order. A special type of sequence is a Geometric Progression (GP), where you get from one number to the next by multiplying by the same number every time. We call this number the common ratio (r).

Example: The sequence 2, 6, 18, 54... is a GP because you multiply by 3 each time. Here, r=3.

Question 1.1: If the first term of a GP is 4 and the common ratio is 2, what is the 5th term?
Show Solution

Term 1: 4

Term 2: 4 * 2 = 8

Term 3: 8 * 2 = 16

Term 4: 16 * 2 = 32

Term 5: 32 * 2 = 64. The 5th term is 64.

Notice a pattern? The nth term is the first term multiplied by r, (n-1) times. Formula: aₙ = a * rⁿ⁻¹.

Phase 2: A Smarter Way to Write Terms

Instead of writing out the whole sequence, we can use the formula aₙ = a * rⁿ⁻¹. Let's use this to connect different terms.

Question 2.1: If the first term is 'a' and ratio is 'r', write the 1st and 5th terms using the formula. What is their product (a₁ * a₅)?
Show Solution

a₁ = a * r¹⁻¹ = a * r⁰ = a

a₅ = a * r⁵⁻¹ = ar⁴

Product: a₁ * a₅ = a * ar⁴ = a²r⁴ = (ar²)²

Interesting! Notice that ar² is the 3rd term (a₃). So, a₁ * a₅ = a₃². This is a useful shortcut!

Phase 3: Finding Unknowns from Clues

Now let's use the ideas from Phase 2 to solve a puzzle. This is the core skill for the JEE problem.

Question 3.1: In a GP, we are told that the product of the 1st and 5th term is 49 (a₁a₅ = 49). What is the 3rd term (a₃)?
Show Solution

From our shortcut in Phase 2, we know a₁a₅ = a₃².

So, a₃² = 49.

Taking the square root, a₃ = 7 (assuming positive terms).

Question 3.2: We are also told the sum of the 2nd and 4th term is 14.5 (a₂ + a₄ = 14.5). Can you write this using 'a' and 'r'?
Show Solution

a₂ = ar

a₄ = ar³

So, ar + ar³ = 14.5. We can factor this: a(r + r³) = 14.5.

Phase 4: The JEE Main Question

You now have all the skills. The numbers are just a bit different. Try to solve it yourself first!

Question 1: Let a₁, a₂, a₃, . . . be a G.P. of increasing positive terms. If a₁a₅ = 28 and a₂ + a₄ = 29, then a₆ is equal to:
Show Solution

Step 1: Use the product clue.

a₁a₅ = a₃² = 28. So, a₃ = √28 = 2√7.

Step 2: Use the sum clue.

a₂ + a₄ = ar + ar³ = a(r + r³) = 29. This is not as helpful. Let's try expressing it using a₃:

a₂ = a₃/r and a₄ = a₃*r. So, a₃/r + a₃*r = 29.

a₃(1/r + r) = 29.

Step 3: Solve for r.

Substitute a₃: 2√7 * (1/r + r) = 29. Multiply by r: 2√7(1 + r²) = 29r.

This is a quadratic: 2√7r² - 29r + 2√7 = 0. Using the quadratic formula gives r = 2√7 or r = 1/(2√7). Since the terms are "increasing", r must be > 1. So, r = 2√7.

Step 4: Find a₆.

We want a₆. We know a₃. a₆ = a₃ * r³.

a₆ = (2√7) * (2√7)³ = (2√7)⁴ = 2⁴ * (√7)⁴ = 16 * 49 = 784.
