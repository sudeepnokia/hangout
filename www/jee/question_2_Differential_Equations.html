Build Your Skills: Differential Equations

This guide will build your understanding from basic algebra to solving a JEE-level differential equation.

Phase 1: What is a Differential Equation?

A differential equation is just an equation that contains a derivative (like dy/dx). It connects a function with its rate of change. The goal is to find the original function.

The simplest type is a Linear Differential Equation. For our problem, the standard form we are looking for is: dx/dy + P(y)x = Q(y). Here, P(y) and Q(y) are just expressions involving the variable y.

Question 1.1: Take the equation y * (dx/dy) + x = y². Can you rearrange it to match the standard form dx/dy + P(y)x = Q(y)? What are P(y) and Q(y)?
Show Solution

To get dx/dy by itself, we need to divide the entire equation by y.

(y * (dx/dy))/y + x/y = y²/y

dx/dy + (1/y)x = y

This perfectly matches the standard form! By comparing, we can see:

P(y) = 1/y

Q(y) = y

Phase 2: The Integrating Factor (IF)

To solve equations in this form, we use a special tool called the Integrating Factor (IF). It's a function we multiply the whole equation by to make it easy to solve. The formula for it is: IF = e^(∫P(y)dy).

Question 2.1: From Phase 1, we know P(y) = 1/y. Calculate the Integrating Factor.
Show Solution

First, we need to find the integral of P(y):

∫P(y)dy = ∫(1/y)dy = ln(y) (the natural logarithm of y).

Now, we plug this into the IF formula:

IF = e^(ln(y))

Since e and ln are inverse functions, they cancel each other out.

IF = y.

Phase 3: Putting It All Together

Once you have the IF, the solution is found using this template: x * (IF) = ∫(Q(y) * IF)dy + C. The 'C' is a constant of integration that we find later.

Question 3.1: Using your IF from Phase 2 (IF=y) and Q(y) from Phase 1 (Q(y)=y), find the general solution.
Show Solution

x * y = ∫(y * y)dy + C

xy = ∫y²dy + C

Now, we integrate y²:

xy = y³/3 + C

This is called the "general solution". To find the specific answer, we need one more clue.

Phase 4: The JEE Main Question

You have now learned all the steps. The final problem just adds one last clue to find the constant C.

Question 2: Let x = x(y) be the solution of the differential equation y(dx/dy) + (x - y²) = 0. If x(1) = 1, then x(1/e) is:
Show Solution

Step 1: Get the General Solution.

From our previous phases, we know the general solution to y(dx/dy) + x = y² is xy = y³/3 + C.

Step 2: Use the clue to find C.

The clue is x(1) = 1, which means "when y=1, x=1". Let's plug this in:

(1)(1) = (1)³/3 + C

1 = 1/3 + C => C = 1 - 1/3 = 2/3.

So the specific solution is xy = y³/3 + 2/3.

Step 3: Find the final answer.

The question asks for x(1/e), which means "what is x when y = 1/e?".

x * (1/e) = (1/e)³/3 + 2/3

x/e = 1/(3e³) + 2/3

Multiply by e: x = e/(3e³) + 2e/3 = 1/(3e²) + 2e/3.
