Learning Guide: The Bohr Model of the Atom

This guide explains Niels Bohr's model for the atom, which successfully predicted the size of electron orbits in hydrogen-like atoms.

Phase 1: Bohr's Postulates

Niels Bohr proposed a model for the hydrogen atom that explained why electrons don't just spiral into the nucleus. He made a few key assumptions (postulates):

  1. Electrons can only exist in specific, stable orbits (or "stationary states") without radiating energy.
  2. The angular momentum of an electron in one of these orbits is quantized, meaning it can only take on discrete values: L = n(h/2π), where n is a positive integer (1, 2, 3, ...) called the principal quantum number.
  3. Electrons can jump between these orbits by absorbing or emitting a photon of a specific energy.

The integer 'n' defines the energy level or shell. n=1 is the ground state, n=2 is the first excited state, and so on.

Phase 2: The Radius of Bohr Orbits

By combining the ideas of quantized angular momentum with classical mechanics (balancing electrostatic force with centripetal force), we can derive a formula for the radius of the n-th orbit.

Bohr Radius Formula: r_n = n² * a₀ / Z

  • r_n: The radius of the n-th orbit.
  • n: The principal quantum number (1, 2, 3, ...).
  • Z: The atomic number (the number of protons in the nucleus). For hydrogen, Z=1. For helium, Z=2.
  • a₀: The Bohr radius. This is the radius of the very first orbit (n=1) in a hydrogen atom (Z=1). Its value is approximately 5.29 x 10⁻¹¹ m. It serves as a fundamental unit of distance in this model.
Warm-up Question: What is the radius of the ground state (n=1) of a hydrogen atom (Z=1)?

Using the formula:

r₁ = 1² * a₀ / 1 = a₀

This is the definition of the Bohr radius, a₀, which the question refers to as the "radius of first stationary state of hydrogen atom".

Phase 3: The Summit - The JEE Main Question

Original Question: Radius of the first excited state of Helium ion is given as r. a₀ → radius of first stationary state of hydrogen atom. Then r is:

Let's break down the question and identify the variables for the Bohr radius formula: r_n = n² * a₀ / Z.

1. Identify the atom/ion:

The question asks about a "Helium ion" (He⁺). A neutral Helium atom has 2 electrons and 2 protons. A Helium ion, He⁺, has lost one electron, leaving it with just one electron orbiting the nucleus. This makes it a "hydrogen-like" ion, and the Bohr model can be applied to it.

For Helium, the atomic number (number of protons) is Z = 2.

2. Identify the energy state:

The question asks for the radius of the "first excited state".

  • The ground state corresponds to n = 1.
  • The first excited state corresponds to n = 2.

3. Apply the formula:

We need to find the radius r for the He⁺ ion where n=2 and Z=2.

r = r_n = n² * a₀ / Z

r = 2² * a₀ / 2

r = 4 * a₀ / 2

r = 2a₀
