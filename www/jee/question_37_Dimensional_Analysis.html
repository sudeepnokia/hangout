Learning Guide: The Power of Dimensions

This guide will show you how to analyze physical quantities by breaking them down into their fundamental building blocks: Mass (M), Length (L), Time (T), and Electric Current (A).

Phase 1: The Basic Building Blocks

Every physical quantity, no matter how complex it sounds, can be expressed as a combination of a few fundamental dimensions.

QuantitySymbolDimension
Massm[M]
Length (or distance)d, x, L[L]
Timet[T]
Electric CurrentI, i[A]

We write the "dimension" of a quantity using square brackets, like [Force].

Warm-up Question: What are the dimensions of Velocity and Acceleration?

Velocity: Velocity is distance divided by time (e.g., meters per second).

[Velocity] = [Length] / [Time] = [L] / [T] = [L T⁻¹]

Acceleration: Acceleration is the change in velocity divided by time (e.g., meters per second squared).

[Acceleration] = [Velocity] / [Time] = [L T⁻¹] / [T] = [L T⁻²]

Phase 2: A Step Up - Dimensions of Force

We can find the dimensions of more complex quantities by using equations that define them.

Newton's Second Law: Force = Mass × Acceleration

This is our key to finding the dimension of Force.

Question: What is the dimension of Force?

We use the equation and substitute the dimensions we already know:

[Force] = [Mass] × [Acceleration]

[Force] = [M] × [L T⁻²] = [M L T⁻²]

This is a very important dimension to remember!

Phase 3: The Magnetic Field (B)

To find the dimension of the magnetic field (B), we need a formula that connects it to things we already know. The formula for the magnetic force on a current-carrying wire is perfect.

Magnetic Force Law: F = I × L × B

Where F is the force, I is the current, L is the length of the wire, and B is the magnetic field strength.

Question: Using the force law, find the dimension of the magnetic field, [B].

First, rearrange the formula to solve for B:

B = F / (I × L)

Now, let's find the dimensions by substituting the dimensions of Force, Current, and Length:

[B] = [Force] / ([Current] × [Length])

[B] = [M L T⁻²] / ([A] × [L])

The [L] on the top and bottom cancels out:

[B] = [M T⁻² A⁻¹]

Phase 4: The Final Piece - Permeability (μ₀)

The permeability of free space, μ₀, is a constant that relates magnetic fields to the currents that create them. A common formula involving it is Ampere's Law, but an easier one for dimensional analysis comes from the force between two parallel wires:

Force / Length = (μ₀ * I₁ * I₂) / (2 * π * distance)

The numbers (like 2 and π) have no dimensions, so we can ignore them for this purpose.

[Force / Length] = [μ₀] * [Current] * [Current] / [distance]

Question: Find the dimension of permeability, [μ₀].

Rearrange the dimensional equation to solve for [μ₀]:

[μ₀] = ([Force / Length] * [distance]) / ([Current]²)

[μ₀] = (([M L T⁻²] / [L]) * [L]) / [A²]

[μ₀] = ([M T⁻²] * [L]) / [A²]

[μ₀] = [M L T⁻² A⁻²]

Phase 5: The Summit - The JEE Main Question

Original Question: If B is magnetic field and μ₀ is permeability of free space, then the dimensions of (B²/μ₀) is:

We have all the pieces. We just need to combine them.

Step 1: Find the dimension of B².

From Phase 3, we have [B] = [M T⁻² A⁻¹].

To square it, we square each component's exponent:

[B²] = ([M T⁻² A⁻¹])² = [M² T⁻⁴ A⁻²]

Step 2: Find the dimension of B²/μ₀.

From Phase 4, we have [μ₀] = [M L T⁻² A⁻²].

Now we divide [B²] by [μ₀]. When dividing dimensions, we subtract the exponents of the denominator from the exponents of the numerator.

[B²/μ₀] = [M² T⁻⁴ A⁻²] / [M¹ L¹ T⁻² A⁻²]

Let's do it one dimension at a time:

  • M: 2 - 1 = 1 → M¹
  • L: 0 - 1 = -1 → L⁻¹
  • T: -4 - (-2) = -4 + 2 = -2 → T⁻²
  • A: -2 - (-2) = -2 + 2 = 0 → A⁰ (which is 1, so it disappears)

Putting it all together:

[B²/μ₀] = [M¹ L⁻¹ T⁻²]

This looks familiar. It's the dimension of Pressure or Energy Density (Energy/Volume).

Let's check the options. The question asks for the dimension of B/μ₀, not B²/μ₀. Let's re-read.

Correction: The question asks for the dimensions of (B/μ₀). Let's recalculate.

[B/μ₀] = [M T⁻² A⁻¹] / [M L T⁻² A⁻²]

  • M: 1 - 1 = 0 → M⁰
  • L: 0 - 1 = -1 → L⁻¹
  • T: -2 - (-2) = 0 → T⁰
  • A: -1 - (-2) = -1 + 2 = 1 → A¹

Putting it together: [B/μ₀] = [L⁻¹ A].
