Build Your Skills: Set Theory

This guide will build your understanding from basic number theory to solving a JEE-level problem involving sets and coprime numbers.

Phase 1: Understanding the Set B

The problem defines a set B based on pairs of numbers (m, n) from the set A = {1, 2, ..., 10}. The conditions are:

  • m and n are from A.
  • m < n.
  • gcd(m, n) = 1. (gcd means "greatest common divisor". gcd=1 means they are coprime - they share no common factors other than 1).

The question asks for n(B), which is the number of such pairs (m, n).

Phase 2: Counting the Pairs Systematically

Instead of trying to find all pairs at once, let's fix the value of n and count how many valid values of m exist for each n.

Question 2.1: For n=2, what are the possible values of m? Remember, m < 2 and gcd(m, 2) = 1.
Show Solution

The only possible value for m is 1. gcd(1, 2) = 1. So we have one pair: (1, 2).

Question 2.2: For n=3, what are the possible values of m? (m < 3, gcd(m, 3) = 1)
Show Solution

Possible m are 1, 2. gcd(1, 3) = 1. gcd(2, 3) = 1. So we have two pairs: (1, 3), (2, 3).

Question 2.3: For n=4, what are the possible values of m? (m < 4, gcd(m, 4) = 1)
Show Solution

Possible m are 1, 2, 3. gcd(1, 4) = 1. gcd(2, 4) = 2 (not coprime). gcd(3, 4) = 1. So we have two pairs: (1, 4), (3, 4).

Phase 3: A Shortcut for Counting Coprimes

The number of positive integers less than n that are coprime to n is given by Euler's Totient Function, φ(n).

For n=2, φ(2)=1. For n=3, φ(3)=2. For n=4, φ(4)=2. This matches our counts from Phase 2!

So, the total number of pairs is the sum of φ(n) for n from 2 to 10.

Let's list the values of φ(n) for n=1 to 10:

  • φ(1) = 1
  • φ(2) = 1
  • φ(3) = 2
  • φ(4) = 2
  • φ(5) = 4
  • φ(6) = 2
  • φ(7) = 6
  • φ(8) = 4
  • φ(9) = 6
  • φ(10) = 4

Phase 4: The JEE Main Question

Now we can solve the problem by summing the values of Euler's Totient Function.

Question 19: Let A = {1, 2, 3, . . ., 10} and B = {m/n : m, n ∈ A, m < n and gcd(m, n) = 1}. Then n(B) is equal to:
Show Solution

Step 1: Understand that n(B) is the number of coprime pairs (m,n) with m < n.

This is equivalent to summing the number of valid m for each n from 2 to 10.

Number of pairs = φ(2) + φ(3) + φ(4) + φ(5) + φ(6) + φ(7) + φ(8) + φ(9) + φ(10).

Step 2: Sum the values from Phase 3.

Sum = 1 + 2 + 2 + 4 + 2 + 6 + 4 + 6 + 4 = 31.

Alternatively, the sum of φ(k) for k=1 to n is related to the number of pairs. The total number of pairs (m,n) with 1<=m,n<=10 is 100. The number of pairs with m=n is 10. The number of pairs with m!=n is 90. The number of pairs with m1 needs to be subtracted. This is more complex. The summation of totients is the most direct method.
