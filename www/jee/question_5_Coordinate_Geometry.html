Build Your Skills: Coordinate Geometry

This guide will build your understanding from basic coordinate points to solving a JEE-level problem involving reflections and centroids.

Phase 1: What is a Centroid?

The centroid is the "center of mass" of a shape. For a triangle, it's the point where the three medians (lines from a vertex to the midpoint of the opposite side) intersect. The formula is simple: it's just the average of the x-coordinates and the average of the y-coordinates.

For vertices (x₁, y₁), (x₂, y₂), (x₃, y₃), the Centroid is ((x₁+x₂+x₃)/3, (y₁+y₂+y₃)/3).

Question 1.1: Find the centroid of a triangle with vertices at (1, 3), (3, 1), and (2, 4).
Show Solution

x-coordinate of centroid: (1 + 3 + 2) / 3 = 6 / 3 = 2.

y-coordinate of centroid: (3 + 1 + 4) / 3 = 8 / 3.

The centroid is at (2, 8/3).

Phase 2: What is an Image in a Line?

An "image" is a reflection, like looking in a mirror. If you have a point and a line (the mirror), the image is another point on the opposite side, at the same perpendicular distance.

A key property: If you reflect a whole shape (like a triangle), the centroid of the new shape is just the reflection of the old centroid. This is a huge shortcut! We don't need to reflect every point.

Question 2.1: The formula to find the image (α, β) of a point (h, k) in the line ax + by + c = 0 is:
(α - h)/a = (β - k)/b = -2(ah + bk + c) / (a² + b²).
Let's find the image of the point (2, 8/3) in the line x + 2y - 2 = 0. What are a, b, and c?
Show Solution

Comparing x + 2y - 2 = 0 to ax + by + c = 0, we get:

a = 1, b = 2, c = -2.

The point we are reflecting is (h, k) = (2, 8/3).

Phase 3: Calculating the Image

Now we plug all our values into the image formula.

Question 3.1: Calculate the value of the expression -2(ah + bk + c) / (a² + b²) using the values from the previous phases.
Show Solution

-2( (1)(2) + (2)(8/3) - 2 ) / (1² + 2²)

= -2( 2 + 16/3 - 2 ) / (1 + 4)

= -2( 16/3 ) / 5

= -32/15

This value is the common ratio for our formula.

Phase 4: The JEE Main Question

You have all the skills and intermediate values. Now you can solve the full problem.

Question 5: Let the triangle PQR be the image of the triangle with vertices (1, 3), (3, 1) and (2, 4) in the line x + 2y = 2. If the centroid of △PQR is the point (α, β), then 15(α – β) is equal to:
Show Solution

Step 1: Find the centroid of the original triangle.

From Phase 1, the centroid is (h, k) = (2, 8/3).

Step 2: Realize the centroid of PQR is the image of this point.

The centroid of the image triangle (α, β) is the image of (2, 8/3) in the line x + 2y - 2 = 0.

Step 3: Use the image formula to find α and β.

From Phase 3, we know: (α - 2)/1 = (β - 8/3)/2 = -32/15.

Solving for α: α - 2 = -32/15 => α = 2 - 32/15 = (30-32)/15 = -2/15.

Solving for β: (β - 8/3)/2 = -32/15 => β - 8/3 = -64/15 => β = 8/3 - 64/15 = (40-64)/15 = -24/15.

Step 4: Calculate the final expression.

15(α - β) = 15( (-2/15) - (-24/15) ) = 15( 22/15 ) = 22.
