Learning Guide: Matching Periodic Properties and Trends

This guide reviews four key periodic properties and their trends to help you match them correctly.

Phase 1: Defining the Properties

Let's define the four properties from List-II:

  • (I) Ionisation Enthalpy (IE): The minimum energy required to remove the most loosely bound electron from an isolated gaseous atom.
    Trend: Generally increases across a period and decreases down a group.
  • (II) Metallic character: The tendency of an atom to lose electrons and form a positive ion. It's the opposite of electronegativity.
    Trend: Decreases across a period and increases down a group.
  • (III) Electronegativity: The ability of an atom to attract shared electrons in a chemical bond.
    Trend: Increases across a period and decreases down a group.
  • (IV) Ionic radii: The radius of an atom's ion. For isoelectronic species (ions with the same number of electrons), the radius decreases as the nuclear charge (number of protons) increases, because the stronger positive nucleus pulls the same number of electrons in more tightly.

Let's analyze each item in List-I and find its corresponding property from List-II.

Analysis of (A): Al³⁺ < Mg²⁺ < Na⁺ < F⁻

  • Let's find the number of electrons in each ion: Al (13p, 10e), Mg (12p, 10e), Na (11p, 10e), F (9p, 10e).
  • These are all isoelectronic species. They all have 10 electrons.
  • The property being compared is Ionic Radii (IV).
  • The rule for isoelectronic species is that radius decreases as nuclear charge (number of protons) increases.
  • Order of protons: F (9) < Na (11) < Mg (12) < Al (13).
  • Therefore, the order of radii should be: Al³⁺ < Mg²⁺ < Na⁺ < F⁻.
  • This matches the given order perfectly. So, A matches with (IV).

Analysis of (B): B < C < O < N

  • These elements are all in Period 2. The general trend for both Ionisation Enthalpy (I) and Electronegativity (III) is to increase from left to right (B < C < N < O).
  • However, there is a well-known exception. Nitrogen (N) has a stable, half-filled p-orbital (2p³). Oxygen (O) has a 2p⁴ configuration. It is easier to remove an electron from the already paired-up orbital in oxygen than from the stable half-filled orbital in nitrogen.
  • Therefore, the Ionisation Enthalpy (I) of Nitrogen is greater than that of Oxygen.
  • The correct order for IE is B < C < O < N.
  • This matches the given order. So, B matches with (I).

Analysis of (C): B < Al < Mg < K

  • Let's consider the trends. K is in Group 1, Period 4. Mg is Group 2, Period 3. Al is Group 13, Period 3. B is Group 13, Period 2.
  • This order doesn't follow a simple left-to-right or top-to-bottom trend for IE or Electronegativity.
  • Let's consider Metallic Character (II). Metallic character increases down and to the left.
  • K is the most metallic (bottom-left). Mg is more metallic than Al (left of Al). Al is more metallic than B (below B).
  • So the order of increasing metallic character is B < Al < Mg < K.
  • This matches the given order. So, C matches with (II).

Analysis of (D): Si < P < S < Cl

  • These elements are all in Period 3, in order from left to right.
  • Both Ionisation Enthalpy and Electronegativity increase across a period.
  • Let's check for exceptions in IE. P has a half-filled p-orbital (3p³), and S has 3p⁴. So, the IE of P > S. The order for IE would be Si < S < P < Cl. This does not match.
  • Let's check Electronegativity (III). Electronegativity does not have this exception and increases steadily across the period.
  • The correct order for electronegativity is Si < P < S < Cl.
  • This matches the given order. So, D matches with (III).
