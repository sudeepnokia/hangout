Build Your Skills: Definite Integrals

This guide will build your understanding from basic algebra and trigonometry to solving a complex definite integral problem from the JEE Main exam.

Phase 1: Simplifying with Factoring and Trig Identities

Complex expressions can often be simplified. A key trigonometric identity is tan²x + 1 = sec²x.

Question 1.1: Simplify the expression f(x) = 7tan⁸x + 7tan⁶x - 3tan⁴x - 3tan²x by factoring.
Show Solution

First, group similar terms:

f(x) = (7tan⁸x + 7tan⁶x) - (3tan⁴x + 3tan²x)

Factor out the common parts in each group:

f(x) = 7tan⁶x(tan²x + 1) - 3tan²x(tan²x + 1)

Now, we can factor out the (tan²x + 1) term:

f(x) = (7tan⁶x - 3tan²x)(tan²x + 1)

Using the identity tan²x + 1 = sec²x, we get:

f(x) = (7tan⁶x - 3tan²x)sec²x. This is much simpler!

Phase 2: A Powerful Integration Technique

Integration by substitution is a technique we use when the function we're integrating is the result of a chain rule differentiation. We look for a function and its derivative in the integral.

The derivative of tan(x) is sec²(x). Our simplified f(x) has both!

Question 2.1: Let's calculate I₁ = ∫(from 0 to π/4) f(x)dx. Use the substitution u = tanx.
Show Solution

Our integral is I₁ = ∫(from 0 to π/4) (7tan⁶x - 3tan²x)sec²x dx.

Let u = tanx. Then du = sec²x dx.

We also need to change the limits of integration. When x=0, u=tan(0)=0. When x=π/4, u=tan(π/4)=1.

The integral becomes: I₁ = ∫(from 0 to 1) (7u⁶ - 3u²) du.

Now, this is a simple polynomial to integrate:

I₁ = [u⁷ - u³] from 0 to 1.

I₁ = (1⁷ - 1³) - (0⁷ - 0³) = 1 - 1 = 0.

Phase 3: Integration by Parts

When we have a product of functions to integrate, like x * f(x), we often use Integration by Parts. The formula is ∫u dv = uv - ∫v du.

For our problem, I₂ = ∫(from 0 to π/4) x * f(x) dx. We can choose u=x and dv = f(x)dx.

Question 3.1: If we set dv = f(x)dx, what is v?
Show Solution

v = ∫f(x)dx. This is the indefinite integral version of I₁.

From our work in Phase 2, the integral of (7u⁶ - 3u²)du is u⁷ - u³. Substituting back u=tanx, we get:

v = tan⁷x - tan³x.

Question 6: Let for f(x) = 7tan⁸x + 7tan⁶x - 3tan⁴x - 3tan²x, I₁ = ∫(from 0 to π/4) f(x)dx and I₂ = ∫(from 0 to π/4) xf(x)dx. Then 7I₁ + 12I₂ is equal to:
Show Solution

Step 1: Calculate I₁.

From Phase 2, we found that I₁ = 0.

Step 2: Calculate I₂ using Integration by Parts.

I₂ = ∫(from 0 to π/4) x * f(x) dx

Let u=x and dv=f(x)dx. Then du=dx and v = tan⁷x - tan³x.

I₂ = [x(tan⁷x - tan³x)](from 0 to π/4) - ∫(from 0 to π/4) (tan⁷x - tan³x)dx.

The first part is (π/4)(1-1) - 0 = 0.

So, I₂ = -∫(from 0 to π/4) (tan⁷x - tan³x)dx = -∫(from 0 to π/4) tan³x(tan⁴x-1)dx.

I₂ = -∫(from 0 to π/4) tan³x(tan²x-1)(tan²x+1)dx = -∫(from 0 to π/4) tan³x(tan²x-1)sec²x dx.

Substitute u=tanx, du=sec²x dx. Limits are 0 to 1.

I₂ = -∫(from 0 to 1) u³(u²-1)du = -[u⁶/6 - u⁴/4](from 0 to 1) = -(1/6 - 1/4) = 1/12.

Step 3: Calculate the final expression.

7I₁ + 12I₂ = 7(0) + 12(1/12) = 0 + 1 = 1.
