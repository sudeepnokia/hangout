Build Your Skills: Probability and Statistics

This guide will build your understanding from basic probability to calculating the mean and variance of a random variable.

Phase 1: Listing Outcomes and Defining the Random Variable

First, let's list all possible outcomes of tossing a coin three times. Let H be Heads and T be Tails.

The 8 possible outcomes are: HHH, HHT, HTH, THH, HTT, THT, TTH, TTT.

The problem defines a random variable X as "the number of times a tail follows a head". Let's calculate X for each outcome.

Question 1.1: What are the possible values that X can take?
Show Solution

HHH: X=0

HHT: X=1 (the HT pair)

HTH: X=1 (the HT pair)

THH: X=0

HTT: X=1 (the HT pair)

THT: X=1 (the HT pair)

TTH: X=0

TTT: X=0

The possible values for X are 0 and 1.

Phase 2: The Probability of Each Value

Now we create a probability distribution table for X. This table shows the probability of X taking on each of its possible values.

Question 2.1: What is P(X=0) and P(X=1)?
Show Solution

There are 4 outcomes where X=0 (HHH, THH, TTH, TTT). So, P(X=0) = 4/8 = 1/2.

There are 4 outcomes where X=1 (HHT, HTH, HTT, THT). So, P(X=1) = 4/8 = 1/2.

Phase 3: The Mean (μ)

The mean, or expected value, of a random variable is the weighted average of its possible values. The formula is μ = E[X] = Σ [x * P(X=x)].

Question 3.1: Calculate the mean (μ) of our random variable X.
Show Solution

μ = (0 * P(X=0)) + (1 * P(X=1))

μ = (0 * 1/2) + (1 * 1/2) = 1/2.

Phase 4: The Variance (σ²)

The variance measures how spread out the values of a random variable are. The formula is σ² = Var(X) = E[X²] - (E[X])². We already have E[X] (the mean). We need to find E[X²].

E[X²] = Σ [x² * P(X=x)].

Question 4.1: Calculate the variance (σ²) of X.
Show Solution

First, find E[X²]:

E[X²] = (0² * P(X=0)) + (1² * P(X=1)) = (0 * 1/2) + (1 * 1/2) = 1/2.

Now, use the variance formula:

σ² = E[X²] - μ² = 1/2 - (1/2)² = 1/2 - 1/4 = 1/4.

Phase 5: The JEE Main Question

Now you have all the components to solve the final problem.

Question 16: A coin is tossed three times. Let X denote the number of times a tail follows a head. If μ and σ² denote the mean and variance of X, then the value of 64(μ + σ²) is:
Show Solution

Step 1: Find the mean (μ) and variance (σ²).

From Phase 3, we found μ = 1/2.

From Phase 4, we found σ² = 1/4.

Step 2: Calculate the final expression.

μ + σ² = 1/2 + 1/4 = 3/4.

64(μ + σ²) = 64 * (3/4) = 16 * 3 = 48.
