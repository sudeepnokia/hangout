Build Your Skills: Inverse Trigonometric Functions

This guide will build your understanding from basic algebraic identities to finding the range of a complex inverse trigonometric function expression.

Phase 1: A Key Identity

For inverse trigonometric functions, there are a few key identities. One of the most important is: sec⁻¹x + cosec⁻¹x = π/2. This is true for all valid values of x (where |x| ≥ 1).

This identity is our starting point. It simplifies the problem greatly.

Phase 2: Finding a² + b²

This problem uses a common algebraic pattern. If you know the sum of two numbers (a+b), how can you find the sum of their squares (a² + b²)?

We start with the expansion of (a+b)² = a² + b² + 2ab. Rearranging this gives us the tool we need: a² + b² = (a+b)² - 2ab.

Question 2.1: Let a = sec⁻¹x and b = cosec⁻¹x. Use the identity from Phase 1 and the algebraic trick to rewrite the expression (sec⁻¹x)² + (cosec⁻¹x)².
Show Solution

We have a + b = π/2.

The expression is a² + b².

Using our trick: a² + b² = (π/2)² - 2ab = π²/4 - 2(sec⁻¹x)(cosec⁻¹x).

To make it easier to work with, we can substitute cosec⁻¹x = π/2 - sec⁻¹x:

(sec⁻¹x)² + (π/2 - sec⁻¹x)². Let's call y = sec⁻¹x. The expression is y² + (π/2 - y)².

Phase 3: Finding Minimum and Maximum Values

Let's expand the expression from Phase 2: S = y² + (π²/4 - πy + y²) = 2y² - πy + π²/4. This is a quadratic function of y (where y = sec⁻¹x). An upward-opening parabola's minimum value is at its vertex.

The vertex of a parabola ax²+bx+c is at x = -b/(2a).

To find the maximum value, we need to check the endpoints of the domain of y = sec⁻¹x. The range (which is the domain for our quadratic S) of sec⁻¹x is [0, π], excluding π/2.

Question 3.1: Find the minimum value of S = 2y² - πy + π²/4.
Show Solution

The vertex is at y = -(-π) / (2*2) = π/4.

The minimum value is S(π/4) = 2(π/4)² - π(π/4) + π²/4 = 2(π²/16) - π²/4 + π²/4 = π²/8.

Question 3.2: Find the maximum value of S by checking the endpoints of the domain [0, π].
Show Solution

The vertex is at π/4. The endpoints are 0 and π. The furthest point from the vertex will give the maximum value.

At y=0: S(0) = 2(0)² - π(0) + π²/4 = π²/4.

At y=π: S(π) = 2(π)² - π(π) + π²/4 = 2π² - π² + π²/4 = π² + π²/4 = 5π²/4.

The maximum value is 5π²/4.

Phase 4: The JEE Main Question

Now you have all the pieces to solve the final problem.

Question 11: Using the principal values of the inverse trigonometric functions, the sum of the maximum and the minimum values of 16((sec⁻¹x)² + (cosec⁻¹x)²) is:
Show Solution

Step 1: Find the min and max of the inner expression.

From Phase 3, we found that the minimum value of (sec⁻¹x)² + (cosec⁻¹x)² is π²/8 and the maximum value is 5π²/4.

Step 2: Multiply by the constant factor.

Minimum value of the full expression = 16 * (π²/8) = 2π².

Maximum value of the full expression = 16 * (5π²/4) = 20π².

Step 3: Find the sum.

Sum = Minimum Value + Maximum Value = 2π² + 20π² = 22π².
