Learning Guide: Advanced Projectile Motion

This guide breaks down projectile motion, focusing on how to calculate height at specific times.

Phase 1: Splitting the Motion

The key to solving any projectile motion problem is to split the motion into two independent parts: a horizontal (x) part and a vertical (y) part.

  • Horizontal Motion: There is no acceleration (ignoring air resistance). The velocity is constant.
  • Vertical Motion: The object is constantly accelerated downwards by gravity (g).

If a particle is launched with initial speed v₀ at an angle θ to the horizontal:

  • Initial horizontal velocity: v_x = v₀ * cos(θ)
  • Initial vertical velocity: v_y = v₀ * sin(θ)
Warm-up Question: A particle is projected at 60 m/s at an angle of 30°. What are its initial horizontal and vertical velocities?

v₀ = 60 m/s and θ = 30°.

cos(30°) = √3 / 2 and sin(30°) = 1/2.

v_x = 60 * cos(30°) = 60 * (√3 / 2) = 30√3 m/s.

v_y = 60 * sin(30°) = 60 * (1/2) = 30 m/s.

Phase 2: Calculating Height at Any Time

To find the height (y) of the projectile at any time (t), we use the standard kinematic equation for displacement under constant acceleration:

y = v_initial_y * t + (1/2) * a * t²

In this case, the initial vertical velocity is v_y and the acceleration is a = -g (negative because it's downwards).

Height Formula: h(t) = v_y * t - (1/2)gt²

Question: For the particle from Phase 1 (initial v_y = 30 m/s), what is its height after the first second (t=1)? (Use g = 10 m/s²).

This height is called h₀ in the main problem.

h₀ = h(1) = (30 m/s) * (1 s) - (1/2) * (10 m/s²) * (1 s)²

h₀ = 30 - 5 = 25 m.

Phase 3: Time to Reach Maximum Height

At the very peak of its trajectory, the projectile stops moving upwards for an instant before it starts falling. This means its vertical velocity is momentarily zero.

We can find the time it takes to reach this peak (t_peak) using the kinematic equation for velocity:

v_final_y = v_initial_y + a*t

0 = v_y - g * t_peak

Time to Peak Formula: t_peak = v_y / g

Question: For the same particle (initial v_y = 30 m/s), how long does it take to reach its maximum height? (Use g = 10 m/s²).

t_peak = (30 m/s) / (10 m/s²) = 3 seconds.

Phase 4: The Summit - The JEE Main Question

Original Question: A particle is projected at an angle of 30° from horizontal at a speed of 60 m/s. The height traversed by the particle in the first second is h₀ and height traversed in the last second, before it reaches the maximum height, is h₁. The ratio h₀ : h₁ is ___.

We have all the pieces. Let's put them together.

Step 1: Find h₀ (height traversed in the first second).

From Phase 2, we already calculated this. The height at t=1 is 25m. The height at t=0 is 0m. So the height traversed *during* the first second is h(1) - h(0) = 25 - 0 = 25 m.

h₀ = 25 m.

Step 2: Identify the "last second" before maximum height.

From Phase 3, we know the time to reach maximum height is t_peak = 3 seconds.

The "last second" before this is the time interval from t = 2 s to t = 3 s.

Step 3: Calculate h₁ (height traversed in that last second).

h₁ = (height at t=3) - (height at t=2)

First, find the height at t=3 (the maximum height):

h(3) = (30)(3) - (1/2)(10)(3)² = 90 - 5*9 = 90 - 45 = 45 m.

Next, find the height at t=2:

h(2) = (30)(2) - (1/2)(10)(2)² = 60 - 5*4 = 60 - 20 = 40 m.

Now, find the height traversed during that second:

h₁ = h(3) - h(2) = 45 m - 40 m = 5 m.

Step 4: Find the ratio h₀ : h₁.

h₀ / h₁ = 25 / 5 = 5.

The ratio is 5:1.

Alternative approach (Symmetry): Due to the symmetry of projectile motion, the distance covered in the first second of ascent (from t=0 to t=1) is the same as the distance covered in the last second of descent (from t=5 to t=6, since total flight time is 6s). Similarly, the distance covered in the last second of ascent (t=2 to t=3) is the same as the distance covered in the first second of descent (t=3 to t=4). The height traversed from t=2 to t=3 is the same as the distance it falls from rest in one second: s = ut + 1/2gt² = 0 + 1/2(10)(1)² = 5m. This confirms h₁=5m.

Final Answer Verification

Derived Answer: The ratio h₀ : h₁ is 5:1, so the value is 5.
