Phase 1: The Projection of a Vector

The projection of a vector b onto another vector a is the "shadow" that b casts on a. The formula for the projection vector (let's call it c) is:

c = ( (a · b) / |a|² ) * a

where a · b is the dot product and |a| is the magnitude of a.

Question 1.1: Find the projection vector c of b = xi + 4k onto a = i + 2j + 2k.
Show Solution

First, calculate the dot product a · b:

a · b = (1)(x) + (2)(0) + (2)(4) = x + 8.

Next, find the squared magnitude of a:

|a|² = 1² + 2² + 2² = 1 + 4 + 4 = 9.

Now, plug these into the formula:

c = ( (x+8) / 9 ) * (i + 2j + 2k).

Phase 2: Solving for an Unknown

The problem gives us a clue about the magnitude of the sum of two vectors: |a + c| = 7. We can use this to solve for the unknown x.

Question 2.1: Use the condition |a + c| = 7 to find the value of x.
Show Solution

First, let's find the vector a + c:

a + c = a + ( (x+8)/9 )a = (1 + (x+8)/9)a = ((9+x+8)/9)a = ((x+17)/9)a.

Now, let's find its magnitude:

|a + c| = |((x+17)/9)a| = |(x+17)/9| * |a|.

We know |a| = √9 = 3. So, |a + c| = |(x+17)/9| * 3 = |x+17|/3.

We are given that this equals 7: |x+17|/3 = 7 => |x+17| = 21.

This gives two possibilities: x+17 = 21 or x+17 = -21.

x = 4 or x = -38. The problem states x > 0, so we must have x = 4.

Phase 3: The Geometric Meaning of the Cross Product

The magnitude of the cross product of two vectors, |b x c|, gives the area of the parallelogram formed by those two vectors.

Question 3.1: Calculate the area of the parallelogram formed by vectors b and c.
Show Solution

We have b = 4i + 4k (since x=4).

And c = ((4+8)/9)a = (12/9)a = (4/3)a = (4/3)(i + 2j + 2k).

Area = |b x c| = |(4i + 4k) x (4/3)(i + 2j + 2k)|

Area = (4/3) * |(i+k) x (i+2j+2k)| (factoring out 4 from b)

The cross product is: (i+k) x (i+2j+2k) = | i j k | = i(2-0) - j(2-1) + k(2-0) = 2i - j + 2k.

The magnitude of this vector is √(2² + (-1)² + 2²) = √9 = 3.

So, the area is (4/3) * 4 * 3 = 16.
