Phase 1: The Intersection of Two Lines

Two lines in 3D intersect if there is a point (x, y, z) that lies on both lines. We can find this point by setting the general coordinates of each line equal to each other.

L₁: (x-1)/1 = (y-1)/-1 = (z+1)/3 = λ. So, a general point on L₁ is (λ+1, -λ+1, 3λ-1).

L₂: (x-1)/α = (y-2)/1 = (z-3)/4 = μ. So, a general point on L₂ is (αμ+1, μ+2, 4μ+3).

Question 1.1: If the lines intersect, their coordinates must be equal. Set the y and z coordinates equal to each other to find λ and μ.

The intersection point B is found by plugging λ=0 into the coordinates for L₁: B = (1, 1, -1).
