Step-by-Step Path to a JEE Main G.P. Problem

This guide will build your understanding of Geometric Progressions (G.P.) to solve a competitive exam question.

The Original Problem (JEE Main 2025, Q1): Let a₁, a₂, a₃, ... be a G.P. of increasing positive terms. If a₁a₅ = 28 and a₂ + a₄ = 29, then what is a₆?

Phase 1: What is a Geometric Progression (G.P.)?

A G.P. is a sequence of numbers where each term after the first is found by multiplying the previous one by a fixed, non-zero number called the common ratio (r).

Example: In the sequence 2, 6, 18, 54, ..., the first term a₁ is 2. The common ratio r is 3 (since 2*3=6, 6*3=18, and so on).

Question 1.1: What is the common ratio of the G.P.: 5, 10, 20, 40, ...?
Show Solution

To find the common ratio, divide any term by its preceding term.

r = 10 / 5 = 2.

You can check this with other terms: 20 / 10 = 2.

Phase 2: The Formula for the n-th Term

There's a formula to find any term in a G.P. without listing all the terms. It is: aₙ = a₁ * rⁿ⁻¹

Where aₙ is the term you want to find, a₁ is the first term, and r is the common ratio.

Question 2.1: For a G.P. with first term a₁ = 4 and common ratio r = 3, what is the 4th term (a₄)?
Show Solution

Using the formula aₙ = a₁ * rⁿ⁻¹:

a₄ = 4 * 3⁴⁻¹ = 4 * 3³ = 4 * 27 = 108.

Phase 3: The Secret Shortcut - A Key G.P. Property

In any G.P., the product of terms equidistant from the center is always the same. For example, a₁ * a₅ = a₂ * a₄ because the sum of the indices is the same (1+5 = 6 and 2+4 = 6).

Question 3.1: In our original problem, we are given a₁a₅ = 28. What can you deduce about the product a₂a₄?
Show Solution

Because the sum of the indices is equal (1 + 5 = 6 and 2 + 4 = 6), the products of these terms must be equal.

Therefore, a₂a₄ = a₁a₅.

Since we are given a₁a₅ = 28, we now know that a₂a₄ = 28. This is the key insight!

Phase 4: Solving a Simple System of Equations

From the original problem and our deduction in Phase 3, we now have two simple pieces of information:

  1. a₂ + a₄ = 29 (from the problem statement)
  2. a₂a₄ = 28 (from our deduction)

We just need to find two numbers that add up to 29 and multiply to 28.

Question 4.1: Find the values of a₂ and a₄.
Show Solution

Let's call the two numbers x and y. We have x + y = 29 and xy = 28.

We can solve this by thinking about factors of 28 (1, 2, 4, 7, 14, 28). Which pair adds up to 29? The pair is 1 and 28.

So, the two terms are 1 and 28. But which is which? The problem states it is an increasing G.P., which means terms get larger as n increases. Therefore, a₄ must be greater than a₂.

So, a₂ = 1 and a₄ = 28.

Phase 5: Finding the Common Ratio (r)

Now that we know two terms, we can find the common ratio.

Question 5.1: Using a₂ = 1 and a₄ = 28, find the value of r².
Show Solution

We know that to get from one term to the next, we multiply by r. So:

a₃ = a₂ * r

a₄ = a₃ * r = (a₂ * r) * r = a₂ * r²

Now, substitute the values we know:

28 = 1 * r²

So, r² = 28.

Phase 6: The Final JEE Main Question

You have all the pieces. The goal is to find the 6th term, a₆.

Final Question: What is the value of a₆?
Show Solution

We can find a₆ by starting from a term we know (like a₄) and multiplying by the common ratio.

a₅ = a₄ * r

a₆ = a₅ * r = (a₄ * r) * r = a₄ * r²

We know both of these values from our previous steps!

a₄ = 28

r² = 28

Now, calculate the final answer:

a₆ = 28 * 28 = 784.

The final answer is 784.
