Build Your Skills: Matrices and Determinants

This guide will build your understanding from basic determinant properties to solving a complex JEE-level problem involving nested adjugate matrices.

Phase 1: Determinant of a Scaled Matrix

A fundamental property of determinants is how they behave when the matrix is multiplied by a scalar (a constant number).

If A is a square matrix of order n, and k is a scalar, then det(kA) = kⁿ * det(A).

Question 1.1: If A is a 3x3 matrix with det(A) = -2, what is det(3A)?
Show Solution

Here, n=3 and k=3.

det(3A) = 3³ * det(A) = 27 * (-2) = -54.

Phase 2: The Adjugate (or Adjoint) Matrix Properties

The adjugate of a matrix has several important properties. For this problem, we need two key ones for a matrix M of order n:

  1. adj(kM) = kⁿ⁻¹ * adj(M)
  2. adj(adj(M)) = det(M)ⁿ⁻² * M
Question 2.1: Let A be a 3x3 matrix. Simplify the expression adj(3A).
Show Solution

Using property 1, with k=3 and n=3:

adj(3A) = 3³⁻¹ * adj(A) = 3² * adj(A) = 9adj(A).

Question 2.2: Let A be a 3x3 matrix with det(A) = -2. Simplify the expression adj(adj(A)).
Show Solution

Using property 2, with n=3:

adj(adj(A)) = det(A)³⁻² * A = det(A)¹ * A = -2A.

Phase 3: The JEE Main Question

Now we combine these properties to solve the problem step-by-step, working from the inside out.

Question 21: Let A be a square matrix of order 3 such that det(A) = -2 and det(3adj(-6 adj(3A))) = 2ᵐ3ⁿ, m, n > 0. Then 4m + 2n is equal to:
Show Solution

Let's simplify the matrix inside the determinant first. Let's call it X.

X = 3 * adj(-6 * adj(3A))

Step 1: Innermost part. As in Phase 2, adj(3A) = 9adj(A).

Step 2: Next layer. The expression becomes X = 3 * adj(-6 * 9adj(A)) = 3 * adj(-54adj(A)).

Step 3: Apply adj(kM) property. Let M = adj(A), k = -54, n=3.
adj(-54adj(A)) = (-54)³⁻¹ * adj(adj(A)) = (-54)² * adj(adj(A)).

Step 4: Apply adj(adj(M)) property. As in Phase 2, adj(adj(A)) = -2A.

Step 5: Combine steps. The expression from Step 3 becomes (-54)² * (-2A) = -2 * 54² * A.

Step 6: Final matrix form. Substitute this back into the expression for X:
X = 3 * (-2 * 54² * A) = -6 * 54² * A.

Step 7: Calculate the determinant. Now we find det(X).

det(X) = det(-6 * 54² * A). Using the property from Phase 1, the scalar is k = -6 * 54² and n=3.

det(X) = (-6 * 54²)³ * det(A) = (-6)³ * (54)⁶ * (-2)

det(X) = (-216) * (54)⁶ * (-2) = 432 * 54⁶.

Step 8: Prime Factorization.

432 = 2 * 216 = 2 * 6³ = 2 * (2*3)³ = 2⁴ * 3³.

54 = 2 * 27 = 2 * 3³. So, 54⁶ = (2 * 3³)⁶ = 2⁶ * 3¹⁸.

det(X) = (2⁴ * 3³) * (2⁶ * 3¹⁸) = 2¹⁰ * 3²¹.

Step 9: Find m and n and the final answer.

We have 2ᵐ3ⁿ = 2¹⁰3²¹, so m=10 and n=21.

The question asks for 4m + 2n = 4(10) + 2(21) = 40 + 42 = 82.
