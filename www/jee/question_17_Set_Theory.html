Build Your Skills: Set Theory

This guide will build your understanding from basic set partitions to counting the number of equivalence relations on a set.

Phase 1: What is an Equivalence Relation?

An equivalence relation is a way of grouping elements in a set that are similar in some way. It must satisfy three properties: reflexive (a~a), symmetric (if a~b then b~a), and transitive (if a~b and b~c then a~c).

A key idea is that every equivalence relation corresponds to a partition of the set. A partition is a way of breaking the set into non-empty, non-overlapping subsets that cover the whole set.

For the set {1, 2, 3}, we need to find all the ways to partition it.

Phase 2: Listing all Partitions

Let's list all the ways to partition the set S = {1, 2, 3}.

Question 2.2: How many ways can we partition S into two subsets?
Show Solution

The subsets must have sizes that add up to 3, so we need one subset of size 2 and one of size 1.

We can choose the element for the subset of size 1 in 3 ways (either 1, 2, or 3). The other two elements form the other subset.

The partitions are: {{1, 2}, {3}}, {{1, 3}, {2}}, and {{2, 3}, {1}}. There are 3 ways.

Question 2.3: How many ways can we partition S into three subsets?
Show Solution

Each subset must have size 1.

There is only one way: {{1}, {2}, {3}}. This corresponds to the equivalence relation where each element is only related to itself.

Phase 3: The JEE Main Question

The number of non-empty equivalence relations is the total number of partitions of the set. This is also known as the Bell number B(n).

Question 17: The number of non-empty equivalence relations on the set {1, 2, 3} is:
Show Solution

Step 1: Count the number of partitions of each possible size.

From Phase 2, we found:

- Partitions into 1 subset: 1 way

- Partitions into 2 subsets: 3 ways

- Partitions into 3 subsets: 1 way

Step 2: Sum the counts.

Total number of partitions = 1 + 3 + 1 = 5.

Each of these partitions corresponds to a unique non-empty equivalence relation.
