Build Your Skills: Limits and Summations

Phase 1: The Method of Differences (Telescoping Series)

Sometimes, a complex fraction can be broken down into the difference of two simpler fractions. This is a powerful technique for simplifying summations.

The key is to find A and B such that 1/((2r-1)(2r+1)) = A/(2r-1) + B/(2r+1). Multiplying by the denominator gives 1 = A(2r+1) + B(2r-1). By choosing strategic values for r (like 1/2 and -1/2), we can find A and B.

Question 1.1: Find A and B for the expression 1/((2r-1)(2r+1)).
Show Solution

If r = 1/2, 1 = A(2) => A = 1/2.

If r = -1/2, 1 = B(-2) => B = -1/2.

So, 1/((2r-1)(2r+1)) = (1/2) * [1/(2r-1) - 1/(2r+1)].

Phase 2: The Telescoping Sum

When we sum terms that are differences, most of the terms cancel out. This is called a telescoping series.

Question 2.1: Let's find the sum S = Σ(from r=1 to n) 1/((2r-1)(2r+1)).
Show Solution

S = (1/2) * Σ [1/(2r-1) - 1/(2r+1)]

Let's write out the first few terms:

r=1: (1/2) * [1/1 - 1/3]

r=2: (1/2) * [1/3 - 1/5]

r=3: (1/2) * [1/5 - 1/7]

... a lot of terms cancel ...

r=n: (1/2) * [1/(2n-1) - 1/(2n+1)]

The only terms that don't cancel are the very first and the very last.

S = (1/2) * [1 - 1/(2n+1)] = (1/2) * [2n / (2n+1)] = n / (2n+1).

Phase 3: What Happens as n Gets Very Large?

The concept of a limit at infinity is about finding the value a function approaches as the input (n) grows without bound. A simple way to do this for rational functions is to divide every term by the highest power of n in the denominator.

Question 3.1: Find the limit of the sum we found in Phase 2 as n approaches infinity: lim(n->∞) n / (2n+1).
Show Solution

Divide the numerator and denominator by n:

lim(n->∞) (n/n) / ((2n/n) + (1/n)) = lim(n->∞) 1 / (2 + 1/n).

As n gets infinitely large, 1/n becomes 0.

So the limit is 1 / (2 + 0) = 1/2.
