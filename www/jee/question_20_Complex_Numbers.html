Phase 1: The Polar Form of Complex Numbers

A complex number z can be written in polar form as z = r(cosθ + isinθ) = re^(iθ), where r is the magnitude (|z|) and θ is the argument (arg(z)).

The problem states that z₁, z₂, and z₃ are on the circle |z|=1, which means their magnitude is 1. So, we can write them as:

z₁ = e^(iθ₁), z₂ = e^(iθ₂), z₃ = e^(iθ₃)

Phase 2: The Sum of Unimodular Complex Numbers

If a set of complex numbers z₁, z₂, z₃ all have a magnitude of 1, there is a very useful property: |z₁ + z₂ + z₃| = |1/z₁ + 1/z₂ + 1/z₃|. This is because for a unimodular complex number, 1/z = z̄ (the conjugate).

The problem asks for |z₁z₂ + z₂z₃ + z₃z₁|². Let's see if we can relate this to the sum.

Question 2.1: How can we rewrite the expression z₁z₂ + z₂z₃ + z₃z₁ using the sum z₁ + z₂ + z₃?
Show Solution

Let's factor out z₁z₂z₃:

z₁z₂ + z₂z₃ + z₃z₁ = z₁z₂z₃ * (1/z₃ + 1/z₁ + 1/z₂)

Taking the magnitude of both sides:

|z₁z₂ + z₂z₃ + z₃z₁| = |z₁z₂z₃| * |1/z₁ + 1/z₂ + 1/z₃|

Since |z₁|=|z₂|=|z₃|=1, we have |z₁z₂z₃| = 1. So,

|z₁z₂ + z₂z₃ + z₃z₁| = |1/z₁ + 1/z₂ + 1/z₃|

Using the property mentioned above, this is equal to |z₁ + z₂ + z₃|.

Therefore, |z₁z₂ + z₂z₃ + z₃z₁|² = |z₁ + z₂ + z₃|².
