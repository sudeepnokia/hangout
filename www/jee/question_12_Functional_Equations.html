Build Your Skills: Functional Equations

This guide will build your understanding from basic functional equations to solving a JEE-level problem involving a second-order differential equation and area calculation.

Phase 1: A Common Functional Equation

The problem states that f(x+y) = f(x)f(y). This is a very famous functional equation. The function that satisfies this is the exponential function, f(x) = e^(kx) for some constant k.

Question 1.1: If f(x) = e^(kx), what is f'(x)? What is f'(0)?
Show Solution

Using the chain rule, f'(x) = k * e^(kx).

f'(0) = k * e^0 = k * 1 = k.

So, if we know f'(0), we know the function! In our problem, we are given f'(0) = 4a. So, k = 4a, and the function is f(x) = e^(4ax).

Phase 2: Verifying with a Differential Equation

The problem also gives a differential equation: f''(x) - 3af'(x) - f(x) = 0. We can use our function from Phase 1 to solve for the unknown constant 'a'.

Question 2.1: Find f''(x) for f(x) = e^(4ax) and substitute f(x), f'(x), and f''(x) into the differential equation.
Show Solution

f(x) = e^(4ax)

f'(x) = 4a * e^(4ax)

f''(x) = (4a)² * e^(4ax) = 16a² * e^(4ax)

Substituting into the equation:

(16a² * e^(4ax)) - 3a(4a * e^(4ax)) - e^(4ax) = 0

16a²e^(4ax) - 12a²e^(4ax) - e^(4ax) = 0

We can divide by e^(4ax) (since it's never zero):

16a² - 12a² - 1 = 0

4a² = 1 => a² = 1/4. Since a > 0, we have a = 1/2.

Phase 3: Calculating Area with a Definite Integral

The area of a region bounded by a curve y = g(x), the x-axis, and the lines x=c and x=d is given by the definite integral: Area = ∫(from c to d) g(x) dx.

Question 3.1: The problem asks for the area of the region R = {(x, y) | 0 ≤ y ≤ f(ax), 0 ≤ x ≤ 2}. What is the function we need to integrate, and what are the limits?

Show Solution

The upper boundary of the region is y = f(ax). This is the function we need to integrate.

The limits for x are given as 0 to 2.

So, the area is ∫(from 0 to 2) f(ax) dx.

Phase 4: The JEE Main Question

Now you have all the pieces to solve the final problem.

Question 12: Let f : R → R be a twice differentiable function such that f(x + y) = f(x)f(y) for all x, y ∈ R. If f'(0) = 4a and f satisfies f''(x) – 3af'(x) − f(x) = 0, a > 0, then the area of the region R = {(x, y) | 0 ≤ y ≤ f(ax), 0 ≤ x ≤ 2} is:
Show Solution

Step 1: Find the function f(x) and the value of a.

From Phase 1 and 2, we found that a = 1/2 and f(x) = e^(4ax) = e^(2x).

Step 2: Set up the integral for the area.

We need to find the area under the curve y = f(ax) from x=0 to x=2.

f(ax) = f(x/2) = e^(2*(x/2)) = e^x.

Area = ∫(from 0 to 2) e^x dx.

Step 3: Evaluate the integral.

The integral of e^x is just e^x.

Area = [e^x] (from 0 to 2) = e² - e⁰ = e² - 1.
