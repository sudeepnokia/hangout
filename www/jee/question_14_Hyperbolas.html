Build Your Skills: Hyperbolas

This guide will build your understanding from the basic definition of a hyperbola to solving a JEE-level problem involving its foci and latus rectum.

Phase 1: What Defines a Hyperbola?

A hyperbola is defined by its two foci (F₁ and F₂) and a constant difference. For any point P on the hyperbola, the absolute difference of the distances from P to the foci is constant: |PF₁ - PF₂| = 2a, where 'a' is the semi-major axis.

The distance between the foci is 2c, and the center of the hyperbola is the midpoint of the foci.

Question 1.1: The foci of a hyperbola are at (1, 14) and (1, -12). What is the center of the hyperbola? What is the distance between the foci (2c)?
Show Solution

The center is the midpoint: ( (1+1)/2, (14-12)/2 ) = (1, 1).

The distance between the foci is 14 - (-12) = 26. So, 2c = 26, which means c = 13.

Since the x-coordinates of the foci are the same, this is a vertical hyperbola.

Phase 2: Finding the Constant 'a'

We can use the defining property of the hyperbola and a point on it to find the value of 'a'.

Question 2.1: The hyperbola passes through the point P(1, 6). Using the foci F₁(1, 14) and F₂(1, -12), find the value of 2a.
Show Solution

PF₁ = √((1-1)² + (14-6)²) = √(0² + 8²) = 8.

PF₂ = √((1-1)² + (-12-6)²) = √(0² + (-18)²) = 18.

2a = |PF₁ - PF₂| = |8 - 18| = 10. So, a = 5.

Phase 3: Finding 'b' and the Latus Rectum

For any hyperbola, the values a, b, and c are related by the identity: c² = a² + b².

The length of the latus rectum of a hyperbola is given by the formula 2b²/a.

Question 3.1: Using the values of a and c from the previous phases, find b².
Show Solution

We have a = 5 and c = 13.

13² = 5² + b²

169 = 25 + b²

b² = 144.

Phase 4: The JEE Main Question

Now you have all the components to solve the final problem.

Question 14: Let the foci of a hyperbola be (1, 14) and (1, -12). If it passes through the point (1, 6), then the length of its latus-rectum is:
Show Solution

Step 1: Find a, b, and c.

From Phase 1, we found the center is (1,1) and c = 13.

From Phase 2, we used the point (1,6) to find a = 5.

From Phase 3, we used the identity c²=a²+b² to find b² = 144.

Step 2: Calculate the length of the latus rectum.

The formula is 2b²/a.

Length = 2 * 144 / 5 = 288/5.
