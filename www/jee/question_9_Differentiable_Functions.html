Build Your Skills: Differentiable Functions

This guide will build your understanding from basic differentiation to solving a JEE-level problem involving functional equations.

Phase 1: The Definition of a Derivative

The derivative of a function f(x), written as f'(x), is defined as: f'(x) = lim(h->0) [f(x+h) - f(x)] / h. This represents the instantaneous rate of change of the function.

Question 1.1: A function satisfies the equation f(x+y) = f(x) + f(y). Find f'(x) using the definition of the derivative.
Show Solution

f'(x) = lim(h->0) [f(x+h) - f(x)] / h

Using the given property, f(x+h) = f(x) + f(h).

f'(x) = lim(h->0) [f(x) + f(h) - f(x)] / h = lim(h->0) f(h) / h.

Also, f'(0) = lim(h->0) [f(0+h) - f(0)] / h = lim(h->0) f(h)/h (since f(0)=0 from the original property).

So, f'(x) = f'(0), which means the derivative is a constant.

Phase 2: Differentiating a Functional Equation

The given problem has a more complex functional equation: f(x+y) = f(x)f'(y) + f'(x)f(y). This looks like the sine addition formula: sin(a+b) = sin(a)cos(b) + cos(a)sin(b). This suggests that f(x) might be related to sin(kx) and f'(x) to cos(kx).

Let's try to find f'(x) by differentiating the functional equation with respect to x (treating y as a constant).

Question 2.1: Differentiate f(x+y) = f(x)f'(y) + f'(x)f(y) with respect to x.
Show Solution

f'(x+y) = f'(x)f'(y) + f''(x)f(y).

Now, let x=0. f'(y) = f'(0)f'(y) + f''(0)f(y).

Let's try differentiating with respect to y (treating x as a constant).

f'(x+y) = f(x)f''(y) + f'(x)f'(y).

Comparing the two expressions for f'(x+y), we get: f'(x)f'(y) + f''(x)f(y) = f(x)f''(y) + f'(x)f'(y).

f''(x)f(y) = f(x)f''(y) => f''(x)/f(x) = f''(y)/f(y). This means that this ratio is a constant, let's say k.

So, f''(x) = kf(x). This is a second-order linear differential equation.

Phase 3: Solving f''(x) = kf(x)

The solution to this differential equation depends on the sign of k. Let's find k.

From f'(y) = f'(0)f'(y) + f''(0)f(y), we have f''(0)f(y) = f'(y)(1-f'(0)). So f''(0)/f(y) = (1-f'(0))/f'(y). This doesn't seem to lead anywhere.

Let's try another approach. Let's assume f(x) = A*sin(ax+b). Then f'(x) = A*a*cos(ax+b). f(0)=1 => A*sin(b)=1. f(x+y) = A*sin(a(x+y)+b). f(x)f'(y)+f'(x)f(y) = A*sin(ax+b)*A*a*cos(ay+b) + A*a*cos(ax+b)*A*sin(ay+b) = A²a*sin(a(x+y)+2b). This is not working.

Let's go back to the original equation and set y=0. f(x) = f(x)f'(0) + f'(x)f(0). We are given f(0)=1. So, f(x) = f(x)f'(0) + f'(x). f'(x) = f(x)(1-f'(0)). Let 1-f'(0) = k. Then f'(x) = kf(x). This is a first-order linear differential equation.

dy/dx = ky => dy/y = kdx => ln|y| = kx + C => y = A*e^(kx). So f(x) = A*e^(kx).

We are given f(0)=1, so A*e^0 = 1 => A=1. So f(x) = e^(kx).

Then f'(x) = k*e^(kx). f'(0) = k. From 1-f'(0)=k, we get 1-k=k => 2k=1 => k=1/2.

So, f(x) = e^(x/2).

Phase 4: The JEE Main Question

Now we have the function, we can solve the final problem.

Question 9: Let f(x) be a real differentiable function such that f(0) = 1 and f(x + y) = f(x)f'(y) + f'(x)f(y) for all x,y ∈ R. Then Σ(from n=1 to 100) logₑf(n) is equal to:
Show Solution

Step 1: Find the function f(x).

From our work in Phase 3, we found that f(x) = e^(x/2).

Step 2: Find logₑf(n).

logₑf(n) = logₑ(e^(n/2)) = n/2.

Step 3: Calculate the summation.

We need to find Σ(from n=1 to 100) n/2.

= 1/2 * Σ(from n=1 to 100) n.

The sum of the first N integers is given by N(N+1)/2.

= 1/2 * (100 * 101 / 2) = 1/2 * 5050 = 2525.
