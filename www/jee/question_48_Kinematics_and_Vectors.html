Learning Guide: Advanced Vector Kinematics

This guide covers how to work with position, velocity, and angular momentum using vectors and calculus.

Phase 1: From Position to Velocity

In physics, an object's position can be described by a vector, often changing with time. For example, r(t) = x(t)î + y(t)ĵ + z(t)k̂.

To find the velocity vector v(t), we simply differentiate the position vector with respect to time. This means differentiating each component separately.

Velocity Vector: v(t) = dr/dt = (dx/dt)î + (dy/dt)ĵ + (dz/dt)k̂

Remember the power rule for differentiation: d/dt (atⁿ) = n*atⁿ⁻¹.

Warm-up Question: The position of a particle A is given by r_A = (t²î + 3ntĵ + 2tk̂) m. What is its velocity vector, v_A(t)?

We differentiate each component of r_A with respect to t:

  • d/dt (t²) = 2t
  • d/dt (3nt) = 3n
  • d/dt (2t) = 2

So, the velocity vector is: v_A(t) = (2tî + 3nĵ + 2k̂) m/s.

Phase 2: Orthogonal Vectors and Magnitudes

Two vectors are orthogonal (perpendicular) if their dot product is zero.

If A = A_xî + A_yĵ + A_zk̂ and B = B_xî + B_yĵ + B_zk̂, then:

Dot Product: A · B = A_x*B_x + A_y*B_y + A_z*B_z

The magnitude (or length) of a vector A, written as |A|, is found using the Pythagorean theorem in 3D:

Magnitude: |A| = √(A_x² + A_y² + A_z²)

Question: The velocities of two particles at t=1s are v_A = (2î + 3nĵ + 2k̂) and v_B = (2î - t²ĵ + 4pk̂). If their velocities are orthogonal at t=1s, what is the value of p?

First, find v_B at t=1s: v_B(1) = (2î - 1ĵ + 4pk̂).

The condition for orthogonality is v_A · v_B = 0.

(2)(2) + (3n)(-1) + (2)(4p) = 0

4 - 3n + 8p = 0. We can't solve this yet, as it has two unknowns. This shows we often need multiple pieces of information.

Phase 3: Angular Momentum of a Particle

Angular momentum (L) is the rotational equivalent of linear momentum. For a single particle, it is defined with respect to a reference point (origin).

Angular Momentum Formula: L = r x p

  • L is the angular momentum vector.
  • r is the position vector from the reference point to the particle.
  • p is the linear momentum vector (p = mv).
  • The 'x' denotes the cross product.

The question asks for the angular momentum of particle A with respect to the position of particle B. This means we must use the relative position vector: r_relative = r_A - r_B.

So, the formula becomes: L = (r_A - r_B) x p_A

Phase 4: The Summit - The JEE Main Question

Original Question: The position vectors of two 1 kg particles, (A) and (B), are given by r_A = (t²î + 3ntĵ + 2tk̂) m and r_B = (2tî + t²ĵ + 4ptk̂) m, respectively; (constants n, p). At t = 1 s, |v_A| = |v_B| and velocities v_A and v_B of the particles are orthogonal to each other. At t = 1 s, the magnitude of angular momentum of particle (A) with respect to the position of particle (B) is √L kgm²/s. The value of L is ___.

Step 1: Find velocity vectors and evaluate at t=1s.

v_A(t) = dr_A/dt = 2tî + 3nĵ + 2k̂ → v_A(1) = 2î + 3nĵ + 2k̂

v_B(t) = dr_B/dt = 2î + 2tĵ + 4pk̂ → v_B(1) = 2î + 2ĵ + 4pk̂

Step 2: Use the orthogonality condition to relate n and p.

v_A(1) · v_B(1) = 0

(2)(2) + (3n)(2) + (2)(4p) = 0

4 + 6n + 8p = 0 → 2 + 3n + 4p = 0 (Equation 1)

Step 3: Use the equal magnitude condition to find n.

|v_A(1)| = |v_B(1)| → |v_A(1)|² = |v_B(1)|²

2² + (3n)² + 2² = 2² + 2² + (4p)²

4 + 9n² + 4 = 4 + 4 + 16p²

9n² = 16p² → 3n = ±4p. Let's test both possibilities with Eq 1.

Case 1: 3n = 4p. Substitute into Eq 1: 2 + (4p) + 4p = 0 → 8p = -2 → p = -1/4. Then 3n = 4(-1/4) = -1 → n = -1/3.

Case 2: 3n = -4p. Substitute into Eq 1: 2 + (-4p) + 4p = 0 → 2 = 0. Impossible. So we must use Case 1.

We found n = -1/3 and p = -1/4.

Step 4: Find all vectors at t=1s with these values.

r_A(1) = 1î + 3(-1/3)ĵ + 2k̂ = î - ĵ + 2k̂

r_B(1) = 2î + 1ĵ + 4(-1/4)k̂ = 2î + ĵ - k̂

v_A(1) = 2î + 3(-1/3)ĵ + 2k̂ = 2î - ĵ + 2k̂

Mass m_A = 1 kg, so momentum p_A(1) = m_A * v_A(1) = 2î - ĵ + 2k̂.

Step 5: Calculate the relative position vector.

r_A/B = r_A(1) - r_B(1) = (î - ĵ + 2k̂) - (2î + ĵ - k̂) = -î - 2ĵ + 3k̂

Step 6: Calculate the angular momentum (L) using the cross product.

L = r_A/B x p_A

L = (-î - 2ĵ + 3k̂) x (2î - ĵ + 2k̂)

Using the determinant method for cross product:

L = î((-2)(2) - (3)(-1)) - ĵ((-1)(2) - (3)(2)) + k̂((-1)(-1) - (-2)(2))

L = î(-4 + 3) - ĵ(-2 - 6) + k̂(1 + 4)

L = -î + 8ĵ + 5k̂

Step 7: Find the value of L from the magnitude.

The magnitude of angular momentum is |L| = √((-1)² + 8² + 5²) = √(1 + 64 + 25) = √90.

The problem states the magnitude is √L.

So, √L = √90, which means L = 90.

Final Answer Verification

Derived Answer: The value of L is 90.
